# Simple Tweet 專案

### 功能說明

- 社群網站，推文、回覆、點擊按讚與留言。
- 使用者及其他使用者個人頁面。
- 管理者查看全部推文及使用者清單。

### 開發工具

- react 版本 18.2.0
- axios 版本 0.27.2

### 專案網址

https://syuyuyu.github.io/simple-twitter/

### 本地安裝

1.確認本機環境已下載 npm 及 Node.js

2.clone 專案至本機

```
git clone https://github.com/syuyuyu/simple-twitter.git
```

3.開啟終端機(Terminal)進入專案資料夾，輸入

```
npm install
```

4.終端機輸入開啟專案

```
npm start
```

5.瀏覽器開啟畫面成功

### 運行

1.管理者測試帳號密碼設定

```
帳號 : root ,密碼 : 12345678
```

2.使用者測試帳號密碼設定

```
帳號 : user1 ,密碼 : 12345678
```
