# Simple Tweet 專案

## 專案功能

1.使用者

- 註冊、登入，成功登入與錯誤提示。
- 發佈推文、回覆留言功能。
- 點讚、追隨功能。
- 熱門追蹤 top10。
- 點擊頭像連結其他使用者個人頁面。
- 修改個人資料。
- 上傳大頭照、背景圖、個人簡介。
- 追隨中及追隨者頁面。

  2.管理者

- 觀看所有推文清單。
- 刪除推文功能。
- 觀看所有使用者清單。

### 專案網址

https://syuyuyu.github.io/simple-twitter/

### 本地安裝

1.確認本機環境已下載 npm 及 Node.js

2.clone 專案至本機

```
git clone https://github.com/syuyuyu/simple-twitter.git
```

3.開啟終端機(Terminal)進入專案資料夾，輸入

```
npm install
```

4.終端機輸入開啟專案

```
npm start
```

5.瀏覽器開啟畫面成功

### 運行

1.管理者測試帳號密碼設定

```
帳號 : root ,密碼 : 12345678
```

2.使用者測試帳號密碼設定

```
帳號 : user1 ,密碼 : 12345678
```
